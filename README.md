Assignment done on Windows 11 with C++ on Visual Studio Code. Assignment completed alone.

Commands available: BUY, SELL,DEPOSIT, BALANCE, LIST, QUIT, SHUTDOWN, LOGIN, LOGOUT, WHO, LOOKUP.

Running Instructions: Login into UMD server. Type ./server into one terminal to start the server and ./client 127.0.0.1 4552 in another terminal to create a connection. Do the ./client command again in another terminal for simultaneous connections. If wanting to compile any file (including multiple amounts), type make.
